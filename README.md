## 📋 ¿Qué hace esta app?

Colorfly Studio te permite generar paletas de colores aleatorias con un solo clic. Puedes elegir cuántos colores quieres, verlos en formato HEX o HSL, bloquear los que te gusten y guardar tus paletas favoritas.

---

## 🚀 Cómo usar el demo

Abre el demo en tu navegador desde este link:

**[https://milowoxd.github.io/ProyectoM1_AndresManrique-/](https://milowoxd.github.io/ProyectoM1_AndresManrique-/)**

### Paso a paso

**1. Genera una paleta**
Haz clic en el botón **✨ GENERAR PALETA** o presiona la **barra espaciadora** para crear una nueva paleta aleatoria.

**2. Elige el tamaño**
Usa los botones **6 · 8 · 9** para seleccionar cuántos colores quieres en tu paleta.

**3. Cambia el formato**
Alterna entre **HEX** y **HSL** según el formato que necesites para tu proyecto.

**4. Copia un color**
Haz clic sobre cualquier tarjeta de color o en el botón 📋 para copiar su código al portapapeles automáticamente.

**5. Bloquea colores**
Haz clic en 🔓 para bloquear un color que te guste. Al generar una nueva paleta, ese color se mantendrá fijo.

**6. Guarda tu paleta**
Haz clic en **💾 GUARDAR** para guardar la paleta actual. Puedes ver todas tus paletas guardadas en **📌 GUARDADAS**.

**7. Restaura una paleta**
Dentro del modal de paletas guardadas, haz clic en **↺ RESTAURAR** para cargar una paleta anterior.

## 💻 Cómo ejecutarlo localmente

### Opción 1 — Abrir directo en el navegador (más fácil)

1. Descarga el repositorio como ZIP desde GitHub:
   - Ve a [github.com/Milowoxd/ProyectoM1_AndresManrique-](https://github.com/Milowoxd/ProyectoM1_AndresManrique-)
   - Clic en **Code → Download ZIP**
2. Descomprime el archivo
3. Abre el archivo `index.html` directamente en Firefox o Chrome

### Opción 2 — Clonar con Git

Asegúrate de tener [Git](https://git-scm.com/) instalado, luego ejecuta:

```bash
git clone https://github.com/Milowoxd/ProyectoM1_AndresManrique-.git
cd ProyectoM1_AndresManrique-
```

Abre `index.html` en tu navegador.

### Opción 3 — Servidor local con Python

```bash
python -m http.server 8000
```

Luego abre `http://localhost:8000` en tu navegador.
